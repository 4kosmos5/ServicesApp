# **ServiceApp**

ServiceApp - это тестовое приложение разработано для демонстрации основных концепций и использования различных типов сервисов в Android.

## **Описание**
Каждый класс содержит пример использования соответствующего типа сервиса:

* MyService: Пример использования базового сервиса (Service), который может выполнять операции в фоновом режиме без привязки к конкретной активности.
* MyForegroundService: Демонстрация использования сервиса, который работает в режиме foreground, что обеспечивает стабильную работу даже при сворачивании приложения пользователем.
* MyIntentService: Пример использования IntentService для выполнения асинхронных операций в фоновом режиме.
* MyJobService: Показывает использование JobService для выполнения асинхронных задач с учетом требований к ресурсам устройства.
* MyJobIntentService: Демонстрация JobIntentService, который предоставляет удобный способ выполнения фоновых задач, совместимый с новыми требованиями к выполнению задач в фоновом режиме.
* MyWorker: Использование WorkManager для планирования и выполнения отложенных и периодических задач в фоновом режиме, учитывая ограничения системы.
